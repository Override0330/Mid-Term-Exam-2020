#### 搜索

* 请求url：/search

* 请求方式：GET

* 请求参数

  | 参数     | 参数说明                                                     |
  | -------- | ------------------------------------------------------------ |
  | keywords | 必选，关键词                                                 |
  | limit    | 可选，搜索结果数量限制，默认为30                             |
  | type     | 可选，默认为1（单曲）。取值意义 : 1: 单曲, 10: 专辑, 100: 歌手, 1000: 歌单, 1002: 用户, 1004: MV, 1006: 歌词, 1009: 电台, 1014: 视频, 1018:综合 |
